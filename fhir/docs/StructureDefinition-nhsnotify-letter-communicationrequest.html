Resource Profile: Letter CommunicationRequest

Official URL: https://nhsdigital.github.io/nhs-notify-standards/fhir/StructureDefinition/nhsnotify-letter-communicationrequest Version: 0.1.0

Request to print and post a patient letter.

Defines the structure of a letter submission request to NHS Notify print-and-post service.

Usages:

  • Examples for this Profile: CommunicationRequest/ExampleLetterCommunicationRequest
